Trading Bot

Project 1 (Independent Project)

This is a cryptocurrency trading bot I coded using Python, HTML, CSS, and JavaScript. I decided to code a bot that would automate my actions even when I was away from the trading screen. The bot maintains one sell price or one buy price at a given time. The difference between the current price and the buy/sell price never exceeds a pre-determined "gap". The bot's performance compared to the the simple buy-and-hold strategy is mixed. It does well in markets with large movements and does poorly in sideways markets. The loss incurred when executing a market order also plays a signficant role. There is a Python version that uses actual money and Python and web versions that uses paper money (fake money). Outside packages and APIs involved include Matplotlib, Coinbase Pro API, Cryptocompare API, and Chart.js.

Tic-Tac-Toe Game

Project 2 (Independent Project)

This is a Tic-Tac-Toe game I coded using Python. I already knew the concept behind the game and wanted to create a program that had a good user interface and a bot that would be hard to beat. You can play against three types of opponents: another player, an easy bot, and a hard bot. Playing with another player is just taking your input again. The easy bot randomizes its moves. The hard bot uses the minimax algorithm to find the best possible move. This algorithm is a recursive algorithm that simulates every possible move. On the bot's turn, it picks the move with the highest score, and on your simulated turn, it picks the move with the lowest score. The user interface was built using PyQt5.
